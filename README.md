🛠️ Especificaciones Técnicas del ADN Marroquí
Este contrato valida la interacción entre el recipiente de cocción y la composición química de los ingredientes:

Condensación Térmica (bool): Registra si el plato utiliza el sistema de tapa cónica del Tagine para la recirculación constante del vapor, permitiendo cocciones largas con mínima hidratación externa.

Matriz de Equilibrio (bool): Validador del uso de frutas deshidratadas (ciruelas, albaricoques, pasas) en platos de proteína animal para lograr el perfil dulce-salado.

Especia Base (string): Registro de la mezcla dominante, destacando el Ras el Hanout o el uso de azafrán puro.

Sistema de Feedback: Funciones de darLike y darDislike para la interacción comunitaria on-chain.
